## 📌 Overview

**MediQ** is a modern AI-powered health platform that delivers fast, structured, and accessible medical insights.

It integrates **Llama 3.3 via Groq** with an interactive UI to provide tools for symptom analysis, health tracking, and medical assistance — all in one place.

---

## ⚡ Features

* 💬 AI chatbot for health queries
* 🩺 Symptom checker with structured input
* 📋 Downloadable health reports
* 💊 Prescription explanation in simple language
* 📊 BMI calculator
* ⚡ First aid guidance
* 🧬 Health risk scoring
* 🏥 Hospital locator using OpenStreetMap

## ⚙️ Setup

```bash id="p6r8jk"
git clone https://github.com/riya-varda/mediq-health-platform.git
cd mediq-health-platform

npm install

# create a .env file and add:
VITE_GROQ_API_KEY=your_key_here

npm run dev
```

---

## 🔐 Environment Variable

| Variable          | Description       |
| ----------------- | ----------------- |
| VITE_GROQ_API_KEY | Your Groq API key |
